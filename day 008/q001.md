# myAtoi() Algorithm

A clear step-by-step flow for converting a string to a 32-bit signed integer.

---

## 1. Initial Edge Cases

- If the string is empty → return 0.
- If string is exactly "-" or "+" → return 0.

---

## 2. Setup

- Create an empty result string `res`.
- Iterate through each character in the input string.

---

## 3. Skip Leading Whitespace

- While `res` is empty:

  - Ignore characters that are spaces.

---

## 4. Handle First Valid Character

- If current char is a digit → append to `res`.
- Else if `res` is empty and current char is `+` or `-` → append to `res`.
- Else if `res` is empty and char is invalid → return 0.

---

## 5. Collect Digits

- After the sign (if present):

  - Continue appending digits.
  - Stop when a non-digit appears.

---

## 6. Convert to Integer

- Try converting `res` to an integer.
- If conversion fails → return 0.

---

## 7. 32-bit Clamping

- Define:

  - `INT_MIN = -2^31`
  - `INT_MAX = 2^31 - 1`

- If number < INT_MIN → return INT_MIN.
- If number > INT_MAX → return INT_MAX.

---

## 8. Return Result

- If within bounds, return the parsed integer.
